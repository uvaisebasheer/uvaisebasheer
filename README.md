
# 💫 About Me:
Hi, I'm Uvaise Basheer AKA Uvaise K B 👋

I'm a Senior Full Stack Developer with 8+ years of experience building scalable web applications, cloud-native solutions, and enterprise software using ASP.NET Core, .NET, C#, React, SQL Server, and AWS.
